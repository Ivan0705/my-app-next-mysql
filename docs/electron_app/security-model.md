# Модель безопасности SQL Converter

## Обзор безопасности
SQL Converter разработан с принципом **"Security & Privacy by Design"**.
Мы предлагаем две версии с разными моделями безопасности:

### 1. Electron версия (РЕКОМЕНДУЕТСЯ)
 УРОВЕНЬ БЕЗОПАСНОСТИ: ВЫСОКИЙ
├── 100% локальная обработка
├── SQL НИКОГДА не покидает ваш компьютер
├── Не требует интернет соединения
├── Нативные разрешения файловой системы
└── Полный контроль над данными


### 2. Web версия
 УРОВЕНЬ БЕЗОПАСНОСТИ: СРЕДНИЙ
├── SQL отправляется на наш сервер
├── Обработка в защищенной среде
├── Данные НЕ сохраняются на сервере
├── HTTPS шифрование
└── Подходит для неконфиденциальных данных
